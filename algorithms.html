Algorithms and Data Structures

Artifact Overview

This artifact originates from CS-300: Data Structures and Algorithms. The original application loaded bid data from a CSV file and sorted records using a vector-based sorting implementation.

Enhancement Summary

The artifact was enhanced by implementing a hybrid sorting algorithm. QuickSort was modified to switch to Insertion Sort when processing smaller partitions. This improvement reduced recursive overhead and increased efficiency for small datasets while maintaining the overall performance benefits of QuickSort.
